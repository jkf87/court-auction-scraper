# 법원경매정보 사이트 구조 (courtauction.go.kr)

## 기술 스택
- WebSquare 프레임워크 기반 SPA
- 동적 XML 로딩 → WebFetch로 직접 데이터 접근 불가
- **브라우저 자동화 필수** (Chrome DevTools MCP, OpenClaw browser, Antigravity CDP)

## ⚡ 성능 병목 포인트
- a11y 트리(스냅샷)가 검색 결과 페이지에서 700+ 노드 → 수만 토큰
- `wait_for`가 전체 스냅샷을 반환 → 컨텍스트 윈도우 소진
- 반복적 스냅샷 요청 → Chrome 렌더러 OOM / CDP 타임아웃 → 탭 크래시
- **해결**: 검색 결과 이후 스냅샷 금지, JS evaluate만 사용, async 일괄 페이지 순회

## 주요 URL
- 메인: `https://www.courtauction.go.kr/pgj/index.on`
- 물건상세검색 (메뉴 클릭 시): `?w2xPath=/pgj/ui/pgj100/PGJ151F00.xml`

## 물건상세검색 폼 구조

### 법원/소재지 선택
- 라디오: "법원/담당계" (기본) | "소재지(지번주소)" | "소재지(새주소)"
- 법원 콤보박스 값 (주요):
  - "대전지방법원", "서울중앙지방법원", "인천지방법원", "수원지방법원"
  - "부산지방법원", "대구지방법원", "광주지방법원" 등
- 담당계 콤보박스: "전체", "경매1계"~"경매12계" 등

### 용도 선택 (3단 콤보박스)
- 대분류: "토지", "건물", "차량및운송장비", "기타"
- 중분류 (건물): "주거용건물", "상업용및업무용", "산업용및기타특수용", "용도복합용"
- 소분류 (주거용건물): "단독주택", "다가구주택", "아파트", "연립주택", "다세대주택", "빌라", "오피스텔" 등

### 가격 필터
- 감정평가액: 최소/최대 콤보박스
- 최저매각가격: 최소/최대 콤보박스
- 값: "전체", "1천만원", "5천만원", "1억원", "1억5천만원", ... "10억원"

### 기타 필터
- 입찰구분: "기일입찰" (기본) | "기간입찰" | "전체"
- 기간: 시작/종료 날짜 (기본 오늘~2주후)
- 면적, 유찰횟수, 최저매각가율, 특이사항

## 검색 결과 그리드 구조

### 테이블 ID
`mf_wfm_mainFrame_grd_gdsDtlSrchResult_body_table`

### 행 구조
- 헤더: 2행 (rowspan 사용)
- 데이터: **2행 1세트** (`tr.grid_body_row`)
  - 1행: 체크박스, 사건번호(법원+번호), 물건번호, 소재지+상세, 지도버튼, 비고, 감정평가액, 담당계+매각기일
  - 2행: 용도, 최저매각가격+가율, 진행상태

### 페이지네이션
- 페이지 버튼 ID: `mf_wfm_mainFrame_pgl_gdsDtlSrchPage_page_N` (N=1,2,3...)
- 페이지당 건수: 콤보박스 ID `mf_wfm_mainFrame_sbx_pageSize` (10/20/30/40)
- **40건으로 설정하면 페이지 수를 줄여 효율적**
- 페이지 버튼은 **최대 10개만 표시** (1-10, 11-20 등 그룹 단위)
- "이전 목록" / "다음 목록" 버튼으로 그룹 간 이동
- 실제 총 페이지 수는 `Math.ceil(총물건수 / 페이지크기)`로 계산

### 입찰구분 라디오 버튼
- "기일입찰" (기본값), "기간입찰", "전체"
- description: "기일입찰", "기간입찰", "전체"
- 클릭이 안 되면 evaluate_script로 직접 클릭

## 폼 요소 UID 매핑 (take_snapshot 기준)

검색 폼은 페이지 로드마다 UID가 변경되므로, **fill 시 combobox의 description으로 식별**:
- 법원: description="법원 선택"
- 담당계: description="담당계 선택"
- 대분류: description="대분류 선택"
- 중분류: description="중분류 선택"
- 소분류: description="소분류 선택"
- 감정평가액 최소: description="감정평가액 최소금액 선택"
- 감정평가액 최대: description="감정평가액 최대금액 선택"
- 검색 버튼: description="부동산 물건상세 검색 버튼"

## 주의사항
- 법원을 "전체"로 선택 시 감정평가액/최저매각가격/용도 중 하나 이상 필수
- 기일입찰은 현재~2주 후까지만 검색 가능
- 대전지방법원 관할: 대전광역시 + 세종특별자치시 + 충남 일부(금산, 논산 등)
